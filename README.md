# Indie Hacker Resources

A curated collection of source-linked API buying guides for solo developers building profitable software.

## What's Inside

### [AI & LLM API Landscape](./ai-llm-api-landscape.md)
LLMs, embeddings, speech, image/video, agents, scraping, and AI app stack picks.

### [Communication & Notification APIs](./communication-notification-apis.md)
Email, SMS, WhatsApp, push, in-app notifications, chat, webhooks.

### [Data, Storage & Infrastructure APIs](./data-storage-infrastructure-apis.md)
Databases, vector DBs, storage, cache/KV, search, jobs, compute, flags.

### [Payment & Billing APIs](./payment-billing-apis.md)
Processors, merchant-of-record platforms, usage billing, payouts, tax, and SaaS finance tools.

## Website

The Next.js site lives in `website/`.

- Home: guide discovery + global search
- Guide pages: section TOC + in-guide search + cross-guide links
- Content source: `website/content/*.md`

## Who This Is For

Solo developers and indie hackers who want to:
- Choose APIs faster without bad pricing surprises
- Build SaaS with TypeScript/Next.js
- Skip the research phase and start building
- Know which APIs are actually worth using (and which to avoid)
